# Yunik-GTC
Yunik Game Template for C++

## Overview
This is the template for the game using C++

## Prerequisites
- CMake (>=3.8)

## Required libraries
- OpenGL
- GLFW3
- GLEW
- GLM

## Build

### Windows

#### MinGW

```powershell
> mkdir build
> cd build
> cmake .. -G "MinGW Makefiles"
> mingw32-make
```

#### Visual Studio & Cmake

```powershell
> mkdir build
> cd build
> cmake .. -G "Visual Studio 15 Win64"
> cmake --build . --config Release
```
