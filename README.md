# Yunik-GTC
Yunik Game Template for C++

## Overview
This is the template for the game using C++

## Prerequisites
- CMake
- Ninja
- Clang

## External Dependencies
- OpenGL

## Build

```bash
mkdir build
cd build
cmake .. -G Ninja -DCMAKE_BUILD_TYPE=RELEASE
ninja
```
